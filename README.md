# 前言

我一直对语言模型充满兴趣。十多年前，我通过一门统计模式分类课程开始了我的AI之旅，这促使我完成了第一个独立项目：开发一个模型和网络应用程序，通过歌词来检测歌曲的情绪。

时间来到2022年，随着ChatGPT的发布，大语言模型（LLM）席卷全球，彻底改变了我们许多人的工作方式。这些模型具有极强的通用性，能够帮助完成诸如检查语法、撰写电子邮件、总结冗长文档等任务。这得益于它们解析和生成类人文本的能力，这种能力在从客户服务到内容创作，甚至编程和数据分析等技术领域中都非常重要。

正如它们的名字所暗示的那样，大语言模型（LLM）的一个显著特点是它们“大”——非常大，包含数百万到数十亿的参数。（作为对比，使用更传统的机器学习或统计方法，鸢尾花数据集可以通过仅有两个参数的小模型以超过90%的准确率进行分类。）然而，尽管与更传统的方法相比，LLMs的规模庞大，但它们并不一定是一个黑箱。

在这本书中，你将逐步学习如何构建一个大语言模型（LLM）。到最后，你将从根本上理解像ChatGPT中使用的LLM是如何工作的。我相信，对每个基本概念和底层代码的熟练掌握是成功的关键。这不仅有助于修复错误和提高性能，还能让你尝试新的想法。

几年前，当我开始研究LLM时，我不得不通过艰苦的方式学习如何实现它们，翻阅了许多研究论文和不完整的代码库，才逐渐形成了全面的理解。通过这本书，我希望通过开发和分享一个逐步实现的教程，详细介绍LLM的所有主要组件和开发阶段，让LLM变得更加易于理解。

我坚信，理解LLM的最佳方式是从零开始编写一个模型——而且你会发现，这个过程也可以很有趣！

祝你阅读和编码愉快！

# 关于本书

《从零开始构建大语言模型》这本书旨在帮助你从基础开始理解并创建自己的类GPT大语言模型（LLM）。它首先聚焦于处理文本数据的基本原理和编码注意力机制，然后逐步指导你从零开始实现一个完整的GPT模型。书中还涵盖了预训练机制以及针对特定任务（如文本分类和指令跟随）的微调。通过阅读本书，你将深入理解LLM的工作原理，并掌握构建自己模型的技能。虽然你将创建的模型在规模上比大型基础模型小，但它们使用相同的概念，是理解构建最先进LLM的核心机制和技术的强大教育工具。

## 谁应该读本书

《从零开始构建大语言模型》适合机器学习爱好者、工程师、研究人员、学生和从业者阅读，他们希望深入理解大语言模型（LLM）的工作原理，并学习如何从零开始构建自己的模型。无论是初学者还是经验丰富的开发者，都可以利用他们现有的技能和知识来掌握创建LLM所使用的概念和技术。

这本书的独特之处在于它全面涵盖了构建大语言模型（LLMs）的整个过程，从处理数据集到实现模型架构、在未标注数据上进行预训练，以及针对特定任务进行微调。截至本书撰写时，还没有其他资源提供如此完整且实践性强的从零开始构建LLMs的方法。

要理解本书中的代码示例，你需要扎实掌握Python编程。虽然对机器学习、深度学习和人工智能的一些了解可能有所帮助，但这些领域的深厚背景并非必需。LLMs是人工智能中的一个独特子集，因此即使你是该领域的新手，也能够跟上书中的内容。

如果你对深度神经网络有一定经验，可能会发现某些概念更加熟悉，因为LLMs正是基于这些架构构建的。然而，熟练使用PyTorch并不是前提条件。附录A提供了PyTorch的简明介绍，帮助你掌握理解本书代码示例所需的技能。

高中水平的数学知识，特别是向量和矩阵的运算，在探索LLMs内部工作原理时会有所帮助。然而，掌握本书中呈现的关键概念和思想并不需要高深的数学知识。

最重要的前提条件是扎实的Python编程基础。有了这些知识，你将充分准备好探索LLMs的迷人世界，并理解本书中呈现的概念和代码示例。

## 本书是如何组织的：一份路线图

本书的设计是循序渐进的，每一章都建立在前一章介绍的概念和技术之上。全书分为七章，涵盖了大语言模型（LLM）及其实现的关键方面。

第1章对LLMs背后的基本概念进行了高层次介绍。它探讨了Transformer架构，这是诸如ChatGPT平台上使用的LLMs的基础。

第2章制定了从零开始构建LLM的计划。它涵盖了为LLM训练准备文本的过程，包括将文本分割为单词和子词标记、使用字节对编码进行高级标记化、通过滑动窗口方法采样训练示例，以及将标记转换为输入LLM的向量。

第3章重点介绍LLMs中使用的注意力机制。它引入了一个基本的自注意力框架，并逐步发展到增强的自注意力机制。本章还涵盖了因果注意力模块的实现，使LLM能够一次生成一个标记，通过随机丢弃注意力权重来减少过拟合，并将多个因果注意力模块堆叠成多头注意力模块。

第4章专注于编写一个类似GPT的LLM，可以训练其生成类人文本。它涵盖了诸如归一化层激活以稳定神经网络训练、在深度神经网络中添加快捷连接以更有效地训练模型、实现Transformer块以创建不同大小的GPT模型，以及计算GPT模型的参数数量和存储需求等技术。

第5章实现了LLM的预训练过程。它涵盖了计算训练和验证集损失以评估LLM生成文本的质量、实现训练函数并预训练LLM、保存和加载模型权重以继续训练LLM，以及从OpenAI加载预训练权重。

第6章介绍了不同的LLM微调方法。它涵盖了为文本分类准备数据集、修改预训练的LLM以进行微调、微调LLM以识别垃圾邮件，以及评估微调后的LLM分类器的准确性。

第7章探讨了LLM的指令微调过程。它涵盖了为监督指令微调准备数据集、在训练批次中组织指令数据、加载预训练的LLM并微调它以遵循人类指令、提取LLM生成的指令响应以进行评估，以及评估指令微调后的LLM。
